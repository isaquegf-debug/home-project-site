# Amostra do site Home Project

Landing page institucional criada para validar posicionamento, identidade visual e fluxo inicial de atendimento.

## Abrir

Execute `ABRIR_AMOSTRA.cmd` ou abra `index.html` no navegador.

## Direcao da marca

- Posicionamento: projetos e gestao de solucoes para residencias e empresas.
- Promessa: projetar, planejar, administrar e entregar resultados.
- Paleta: marfim, nogueira, grafite, bronze e verde oliva.
- Imagens: ativos autorais gerados para esta amostra.

## Antes da publicacao

- Confirmar o texto final da regiao de atendimento.
- Incluir projetos reais e depoimentos aprovados quando estiverem selecionados.
- Configurar analytics somente após definir a ferramenta e o consentimento necessário.
- Apontar o dominio `homeproject.com.br` para a hospedagem escolhida.
